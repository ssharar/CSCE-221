Matrix file format:
The first row is the number of rows and columns of the matrix.
From the second row to the end are the triplets of the matrix stored row by row.




Example:

The 3x3 matrix

    1 2 0
    0 1 2
    0 0 1

will be stored as:

3 3
0 0 1
0 1 2
1 1 1
1 2 2
3 3 1
